# CApp Drop Folder

Use this folder for Lab 13 CApp/CAR deployment.

Expected CAR file name used by the Lab 13 commands:

```text
CitizenInfoCompositeExporter_1.0.0.car
```

Expected path from the repository root:

```text
labs/13-wso2-mi-capp-deployment/capps/CitizenInfoCompositeExporter_1.0.0.car
```

Important:

- Lab 12 does not use this CAR file.
- Lab 12 uses the direct Synapse XML ConfigMap path.
- Lab 13 uses this folder after you export a valid `.car` from WSO2 tooling.
- A hand-built CAR reached MI during testing, but MI rejected it with `Some dependencies were not satisfied in cApp`.
